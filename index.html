--color-bray
--color-bray is a variable that defines the bray color.

Bray color background in a div:

div with bray background

--color-bray-light
Light bray is like bray but lighter.

Light bray color background in a div:

div with light bray background

Classes


content-container The content-container class works similar to the container class from Bootstrap.
It sets the left and top padding to var(--size-l) or 3em.

dark The dark class makes the selected element use dark mode.

Usage:

div with dark mode

dot The dot class makes the selected element a dot and it only applies to divs.

Usage:

this is a dot
